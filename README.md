# ❤️ Heart Disease Prediction

A Machine Learning project that predicts the possibility of heart disease using patient health data.

---

## 📌 Project Overview

This project uses Machine Learning algorithms to analyze medical attributes and predict whether a person is likely to have heart disease or not.

The main goal of this project is to understand the complete ML workflow:
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Feature Engineering
- Model Training
- Model Evaluation
- Prediction System

---

## 🚀 Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Streamlit

---

## 📂 Dataset Features

Some important features used in the dataset:

- Age
- Sex
- Chest Pain Type
- Resting Blood Pressure
- Cholesterol
- Fasting Blood Sugar
- Maximum Heart Rate
- Exercise Induced Angina
- ST Depression
- Thalassemia

---

## 🤖 Machine Learning Models Used

- Logistic Regression
- K-Nearest Neighbors (KNN)
- Decision Tree Classifier
- Naive Bayes
